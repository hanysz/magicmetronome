Using multiple tracks

Magic Metronome can play several tracks simultaneously. If you want all the tracks to start at the same time, then you just enclose each track in braces—"{" and "}". For example, here is the code to start two metronomes in two different tempos at the same time:

{ 60 (a) } { 90 (b) }

If you want tracks to start at different times then you need to be a little bit careful. This next example will start a second metronome three beats after the first one:

60 a a a {90(b)} (a)

Notice that the "b" track is in the middle of the "a" track. The rules for multiple tracks are:

  • each "{" will launch a new track;
  • every "{" must have a matching "}" some time later;
  • the new track starts simultaneously with the tick that comes after its "}".
Like parentheses for repeats, tracks can be nested: if you want to do something complicated, you can have a new track starting inside another track.
