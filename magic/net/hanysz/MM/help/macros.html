Macros

Macros allow you to define abbreviations. This can make your script shorter if you're using repetitive patterns, and it can make the structure of a long script clearer. You create a macro with a statement of the form

\name="definition"

and then use it just by typing

\name

The name can be any combination of letters and numbers (but with no punctuation or spaces).

For example, suppose you want to make a track that switches between 3/4 and 4/4 bars in an irregular pattern. You might do something that begins like this:

\34bar="baa"
\44bar="baaa"
\44bar\44bar\34bar\44bar R5(\34bar) \44bar

Or if you had a long piece in AABA form, instead of typing out the A section three times you could use macros:

\Asection="something"
\Bsection="something else"
\Asection\Asection\Bsection\Asection

It's possible to create complicated structures by nesting macros inside other macros. However, a macro can only refer to other macros that were described previously. Therefore you can do this:

\34bar="aaa"
\ten34bars="R10(\34bar)"

but not this:

\firstthing="R10(\secondthing)"
\secondthing="R10(\firstthing)"
